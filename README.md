# Starter Kit — New Project Checklist

## Setup (one time, already done globally)
- [x] OWASP skill at ~/.claude/skills/
- [x] Trail of Bits skills at ~/.claude/skills/

## Per-Project Steps
1. `cp -r ~/Projects/_starter-kit ~/Projects/[new-project-name]`
2. `cd ~/Projects/[new-project-name]`
3. `git init && git add . && git commit -m "chore: project foundation"`
4. Open in VS Code, run `/generate-claude-md` in Claude Code
5. Run `/generate-architecture` in Claude Code
6. Start building with Plan Mode
